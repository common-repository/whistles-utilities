Whistles Utilities is an addon for the Whistles plugin from Justin Tadlock adding an additional widget and menu changes.

== Description ==

> Please note this plugin is no longer maintained.

The plugin adds the following features to the <a href="http://wordpress.org/plugins/whistles/">Whistles plugin</a> which needs to be installed for this plugin work:

* An additional widget to allow you to display the contents of a single Whistle post in a sidebar
* Removes the additional Whistle Groups sub menu item from the Appearance menu and places a link to Whistles Group above the admin page title.

== Installation ==

To install the plugin:

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
